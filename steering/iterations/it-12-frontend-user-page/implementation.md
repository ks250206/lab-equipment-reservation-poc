# 実装内容（イテレーション 12 — フロント User ページとヘッダー認証 UI）

## スコープ

- **`/user` ルート**: ログイン中は `GET /api/users/me` の内容（表示名・メール・ロール・Keycloak ID・登録日）を表示。未ログインはログイン誘導。
- **ヘッダー右**: 未認証は主ボタン風の **ログイン**。認証済みは **マイページ**（`/user` へのリンク、表示名またはメール）と **ログアウト** を並置。

## 非目標

- プロフィール編集（Keycloak 側のため）。
- 新規 API（既存 `/api/users/me` のみ利用）。
