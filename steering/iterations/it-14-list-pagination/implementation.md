# it-14: 装置一覧・装置予約リストのページネーション

## スコープ

- `GET /api/devices` を `items` / `total` / `page` / `page_size` のページング JSON に変更（`page_size` は 20 / 50 / 100、既定 50）。
- `GET /api/devices/{id}/reservations` も同形のページング（窓フィルタは従来どおり）。
- フロント: 装置一覧・装置詳細の予約リストに共通バー（表示件数、最初/前/次/最後、ページ番号入力＋移動）。
- カレンダー表示は窓内の全件を複数ページ取得で結合。
- 開発シード: 各装置に 80 件の予約（UTC 2026-04 付近の連続スロット）を投入し、ページング確認可能にする。

## 非目標

- カーソルベースのページングや無限スクロール。
- `GET /api/reservations`（ユーザー自身の一覧）のページング変更。
