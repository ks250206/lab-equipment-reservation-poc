# 実装内容（イテレーション 17 — 装置カレンダーのドラッグ予約）

## スコープ

- 装置詳細のカレンダー（月・週・日）で、マウスドラッグによる **区間選択** を有効化する。
- FullCalendar が返す `start` / `end`（`end` は排他）をそのまま `POST /api/reservations` の開始・終了に渡す。
- 選択後に **新規予約モーダル**（目的の任意入力・開始終了の確認・変更可）を開き、保存で作成する。
- 既存予約と重なる範囲は選択できないよう `selectOverlap: false` を設定する。

## 非目標

- 既存イベントのドラッグによる日時変更（リサイズ・移動）。
- リスト表示からのドラッグ予約。

## 補足

- `interaction` プラグインは既に読み込み済み。`selectable` / `select` / `selectMirror` を追加する。
