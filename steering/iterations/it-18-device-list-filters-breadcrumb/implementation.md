# 実装内容（イテレーション 18 — 装置一覧の予約フィルターとパンくず）

## スコープ

- `GET /api/devices` に **予約ユーザー名（氏名・メール部分一致）** と **予約が重なる期間（from / to の両方指定時）** のクエリを追加し、該当予約を持つ装置だけに絞り込む。
- 装置一覧に上記の入力 UI（ユーザー名はデバウンス、期間は `datetime-local`）を追加。
- 装置詳細に **パンくず**（装置一覧 > 設置場所 > 装置名）。設置場所は一覧へのリンク（クエリ付き）。
- 装置詳細の **カテゴリ・設置場所** を一覧への `<Link>`（アンカー）で表示。

## 非目標

- 予約フィルター用のファセット集計の追加。
- 期間の片方だけ指定時の挙動（両方あるときのみ期間で絞る）。

## 補足

- `reservation_from` / `reservation_to` は UTC に正規化し、重複判定は既存予約 API と同様の区間交差とする。
